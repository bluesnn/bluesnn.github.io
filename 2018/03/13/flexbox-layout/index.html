flexbox布局速记整理

flexbox 属性集合

父级（外层容器）

flex-direction

属性让你决定Flex项目如何排列，横向（水平）或者纵向（垂直）。

flex-wrap

控制排列是否换行排列

缩写

flex-flow是flex-direction和flex-wrap两个属性的速记属性

justify-content

在主轴方向上对齐方式（横向）

align-items

在交叉轴方向上对齐方式（纵向）

align-content

属性用于多行的Flex容器（纵向））

子级（子容器）

align-self

控制子容器

order

控制子容器

flex

控制项目容器的显示
